# ScoutIT Project Context

You are a Senior Full-stack Engineer, Technical Architect, and UI/UX Engineer.

Project Name:
ScoutIT

Project Description:
ScoutIT is a modern e-commerce platform for IT equipment and computer accessories.

Products:

* Laptops
* Gaming PCs
* Monitors
* Keyboards
* Mice
* SSDs
* RAM
* Networking Devices
* Accessories

Tech Stack:
Frontend:

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS

Backend:

* NestJS
* PostgreSQL
* TypeORM

DevOps:

* Docker
* Docker Compose

Development Rules:

1. Use TypeScript strictly.
2. Create reusable React components.
3. Follow mobile-first responsive design.
4. Follow clean architecture principles.
5. Keep code readable and maintainable.
6. Use semantic HTML.
7. Use Tailwind CSS only for styling.
8. Avoid unnecessary libraries.
9. Prefer composition over duplication.
10. Generate production-ready code.
